# 🚌 GuaguaTime RD - Simulador de Transporte Público

## 📋 Reglas de Cálculo
- **Tiempo Base:** Sumatoria del tiempo de todos los tramos de la ruta.
- **Factor Lluvia:** Aumenta el tiempo estimado en un 40% y suma RD$ 25 al costo.
- **Hora Pico:** Aumenta el tiempo estimado en un 60%.
- **Costo Total:** Suma de los costos de tramos + cargos extras por condiciones.

## 🛠️ Tecnologías utilizadas
- HTML5 Semántico
- CSS3 (Variables, Grid, Flexbox)
- JavaScript Vanilla (Módulos ES6, Fetch API, LocalStorage)

## 🚀 Ejecución
Para probar el proyecto localmente se recomienda usar la extensión **Live Server** en VS Code para evitar errores de CORS con el archivo JSON.
